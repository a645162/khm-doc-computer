# Fcitx5

```bash
sudo pacman -S fcitx5-im fcitx5-chinese-addons fcitx5-material-color fcitx5-configtool
```

## 搜狗输入法

搜狗输入法的fcitx4版本号位4.2.1，反正是4开头。
fcitx5版本号直接从10开头了。

```bash
yay -S fcitx5-sogou fcitx5-pinyin-sougou
```
